# RobinHoodz
RobinHoodz 🧀 is a heist game on Robinhood Chain. Rob the rich, build your AURA, run with a gang, and climb the season leaderboard. Every $FETA trade feeds the Feather-Ral Reserve — a transparent on-chain treasury that pays players real token giveaways, claimed in-game at the First Feather Bank. Rob the rich. Feed the Reserve. Earn by playing.
